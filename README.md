# 🏆 Sports Academy Management System

A simple Sports Academy Management System developed using **Node.js, Express.js, HTML, CSS, and JavaScript**. This project allows users to register players, view player information, search players, and delete player records.

---

## 📌 Features

- 🏃 Register new players
- 👦👧 Support for boys and girls
- 🏏 Multiple sports selection
- 📋 View all registered players
- 🔍 Search players by name or sport
- 🗑 Delete player records
- 💾 Data stored in a JSON file
- 📱 Responsive user interface

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js
- JSON

## ⚙ Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/sports-academy-management-system.git
```

### 2. Open the project folder

```bash
cd sports-academy-management-system
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the server

```bash
node server.js
```

or

```bash
npm start
```
